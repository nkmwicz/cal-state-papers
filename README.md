# Calendar of State Papers Parsing Project

## Generate Data

To generate the data for the Calendar of State Papers Parsing Project, follow these steps:

1. **Clone the Repository**  
   First, clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/nkmwicz/cal-state-papers.git
    cd cal-state-papers
    ```
2. **Set Up the Environment**
3. **Install Dependencies**  
   Make sure you have Python installed. Then, install the required dependencies using pip:
   ```bash
   pip install -r requirements.txt
   ```
4. **Generate the Data**  
   Run the following command to generate the data:
   ```bash
   python get_data.py
   ```
